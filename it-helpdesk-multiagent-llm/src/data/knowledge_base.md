# IT Helpdesk Knowledge Base

## VPN Issues
- Ensure the VPN client (GlobalProtect, Cisco AnyConnect, etc.) is updated to the latest version.
- Restart your system before reconnecting.
- Verify your internet connection is stable and that you can browse external websites.
- If you see "authentication failed," reset your Active Directory password.
- If connection drops repeatedly, switch to a wired connection or different network.

## Password / Access Problems
- Use the self-service password reset portal: [https://passwordreset.company.com](#)
- Passwords must be at least 10 characters with uppercase, lowercase, number, and special character.
- If locked out, contact IT Security to unlock your account.
- MFA code issues: ensure your authenticator app is synced with correct device time.

## Laptop Performance
- Run a full system reboot before submitting a ticket.
- Check background apps (Teams, Chrome tabs) consuming high CPU.
- Ensure disk space > 10GB free.
- Run “Disk Cleanup” and uninstall unused apps.
- Keep antivirus definitions up to date.

## Outlook / Email
- Restart Outlook and ensure it is in “Online Mode.”
- Clear cached credentials from “Windows Credential Manager.”
- Check if mailbox size exceeds 5GB; archive older emails.
- If shared mailbox not syncing, remove and re-add account.

## Network / Connectivity
- Check if you can reach [https://intranet.company.com](#).
- For Wi-Fi, forget and reconnect to the SSID.
- For wired, verify the Ethernet cable is plugged in firmly.
- Flush DNS: `ipconfig /flushdns`.
- Restart router if you’re working from home.

## Software Installation
- Use the Company App Catalog to install approved apps.
- For admin rights, raise an access request ticket.
- Unapproved apps must go through Security review.

## General Support
- Always reboot once before submitting a ticket.
- Add screenshots of errors when possible.
- Use “High Priority” only if you’re completely blocked.
